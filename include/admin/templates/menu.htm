Typ erklärt:
  • Box, dort können Inhalte wie eine Umfrage etc. angezeigt werden.
  • Menü, in ein Menü kann man Menüpunkte anlegen.
  • Menüpunkt wahl, hier kann eine Datei als Menüpunkt gewählt werden.
  • Menüpunkt extern, es kann eine Link angegeben werden der in einem neuen Fenster geöffnet wird.
  • Menüpunkt intern, es kann eine Link angegeben werden der in selben Fenster geöffnet wird.
